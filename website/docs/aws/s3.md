# AWS S3

The AWS S3 (or S3 compatible service) can be used as storage for StoryBooker.

## Install

```sh
npm i storybooker @aws-sdk/client-s3
```

## Usage

```js
import { S3Client } from "@aws-sdk/client-s3";
import { AwsS3StorageService } from "storybooker/adapter/aws-s3";

// Initialize the S3Client
const client = new S3Client(...);
// Create the service adapter
const storage = new AwsS3StorageService(client);

// use as `storage` in StoryBooker options.
```
